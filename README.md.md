# Quant Finance Python Toolkit

10 open-source Python libraries for quantitative finance that don't get enough attention.

Each one with what it does, why it matters, and where to get it.


# gs-quant
*Goldman Sachs' own quant toolkit, open-sourced.* - Pricing, risk analytics, backtesting. The same infrastructure their desks use.

pip install gs-quant

GitHub: [github.com/goldmansachs/gs-quant](https://github.com/goldmansachs/gs-quant)


# Qlib
*Microsoft's AI-first quant investment platform.* - Full ML pipeline from data processing to alpha research to order execution. One framework.

pip install qlib

GitHub: [github.com/microsoft/qlib](https://github.com/microsoft/qlib)

# QF-Lib
*Built at CERN's Pension Fund.* - Created by physicists managing real retirement money. Event-driven backtester with portfolio construction and risk monitoring.

pip install qf-lib

GitHub: [github.com/quarkfin/qf-lib](https://github.com/quarkfin/qf-lib)

# Riskfolio-Lib
*Portfolio optimization beyond Markowitz.*-  Risk parity, CVaR, hierarchical clustering, Black-Litterman. One import, dozens of models.

pip install riskfolio-lib
GitHub: [github.com/dcajasn/Riskfolio-Lib](https://github.com/dcajasn/Riskfolio-Lib)

# skfolio
*Portfolios as ML models.* - Fit. Predict. Cross-validate. Scikit-learn API applied to portfolio construction.

pip install skfolio

GitHub: [github.com/skfolio/skfolio](https://github.com/skfolio/skfolio)

# VectorBT
*Millions of backtest simulations at near-C speed.* - NumPy and Numba under the hood. No Python loops. Just raw performance.

pip install vectorbt

GitHub: [github.com/polakowo/vectorbt](https://github.com/polakowo/vectorbt)

# OpenBB
*The open-source Bloomberg alternative.* - Terminal-style research with normalized data from dozens of providers. Without the $25K/year.


pip install openbb

GitHub: [github.com/OpenBB-finance/OpenBB](https://github.com/OpenBB-finance/OpenBB)

# Pyderivatives
*Options pricing beyond Black-Scholes.* Heston, Kou, Bates models. Implied volatility surfaces and pricing kernels out of the box.

pip install pyderivatives

GitHub: [github.com/jialuechen/pyderivatives](https://github.com/jialuechen/pyderivatives)

# QuantStats
*One line of code = full portfolio tearsheet.* - Sharpe, Sortino, drawdowns, Monte Carlo, monthly heatmap. The fastest way to profile a strategy.

pip install quantstats

GitHub: [github.com/ranaroussi/quantstats](https://github.com/ranaroussi/quantstats)

# fredapi
*Every dataset the St. Louis Fed publishes, straight into pandas.* - Yield curves, CPI, VIX history, employment data. No scraping needed.

pip install fredapi

GitHub: [github.com/mortada/fredapi](https://github.com/mortada/fredapi)

All free. All open source.
